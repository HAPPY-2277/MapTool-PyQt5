# 空间数据质量检查工具

## 软件概述

空间数据质量检查工具是一款运行于单机桌面环境的应用软件，用于对地图数据进行质量检查和错误标记。该工具支持栅格、矢量和文本等多种数据格式，提供可视化的错误标记、记录和查询功能。

---

## 系统要求

- **操作系统**：Windows 10/11
- **运行环境**：Python 3.9+（或使用打包后的exe文件）
- **内存**：建议4GB以上
- **硬盘**：至少100MB可用空间

---

## 支持的数据格式

| 类型 | 格式 | 说明 |
|------|------|------|
| 栅格数据 | .tif, .tiff, .bmp, .png, .jpg, .jpeg | 卫星影像、地图图片等 |
| 矢量数据 | .shp, .geojson | 地理要素数据 |
| 文本数据 | .txt, .csv | 表格数据、坐标数据等 |

## 功能详解

### 一、文件操作

#### 1.1 打开地图数据

**菜单路径**：文件 → 打开地图数据

**快捷键**：Ctrl + O

**操作步骤**：
1. 点击菜单"文件" → "打开地图数据"
2. 在文件选择对话框中选择数据文件
3. 点击"打开"按钮

**支持的文件类型**：
- 栅格文件：GeoTIFF、BMP、PNG、JPEG
- 矢量文件：Shapefile、GeoJSON
- 文本文件：TXT、CSV

#### 1.2 打开错误记录

**菜单路径**：文件 → 打开错误记录

**快捷键**：Ctrl + Shift + O

**操作步骤**：
1. 点击菜单"文件" → "打开错误记录"
2. 选择已保存的错误记录文件（.err.json）
3. 错误记录将加载到错误列表中

#### 1.3 保存错误记录

**菜单路径**：文件 → 保存错误记录

**快捷键**：Ctrl + S

**说明**：
- 保存当前所有错误记录到JSON文件
- 文件名默认与地图文件同名，扩展名为 .err.json
- 可通过"另存为"选择其他保存位置

---

### 二、错误标记

#### 2.1 进入标记模式

**方式一**：点击工具栏"标记错误"按钮

**方式二**：菜单"编辑" → "标记错误"

**快捷键**：M

**标记模式说明**：
- 进入标记模式后，鼠标变为十字光标
- 工具栏"标记错误"按钮呈按下状态
- 再次点击可退出标记模式

#### 2.2 标记错误位置

**栅格/矢量数据**：
1. 进入标记模式
2. 在地图上点击要标记的位置
3. 在弹出的对话框中选择错误类型
4. 输入错误描述信息
5. 点击"确定"保存

**文本数据**：
1. 进入标记模式
2. 点击要标记的文本行
3. 在弹出的对话框中填写错误信息
4. 点击"确定"保存

#### 2.3 错误类型说明

| 错误类型 | 颜色标识 | 说明 |
|----------|----------|------|
| 几何错误 | 红色 | 图形形状、边界等问题 |
| 属性错误 | 绿色 | 属性值缺失、错误等 |
| 拓扑错误 | 蓝色 | 空间关系错误 |
| 坐标错误 | 黄色 | 坐标值异常 |
| 其他 | 灰色 | 其他类型问题 |

---

### 三、错误管理

#### 3.1 查看错误列表

错误列表位于窗口右侧面板，显示所有已标记的错误。

**列表信息包括**：
- 错误编号（自动生成）
- 错误类型
- 错误描述
- 位置坐标
- 处理状态

#### 3.2 错误筛选

在错误面板上方可按条件筛选：

- **按类型筛选**：选择错误类型下拉框
- **按状态筛选**：选择处理状态下拉框

#### 3.3 编辑错误

**方式一**：双击错误列表中的条目

**方式二**：选中错误 → 点击"编辑"按钮

**可修改内容**：
- 错误类型
- 错误描述
- 处理状态
- 备注信息

#### 3.4 删除错误

**方式一**：选中错误 → 点击"删除"按钮

**方式二**：右键点击错误 → 选择"删除"

**注意**：删除操作不可撤销，请谨慎操作。

#### 3.5 定位错误

点击错误列表中的条目，地图视图会自动定位到该错误位置并高亮显示。

---

### 四、错误查询

#### 4.1 打开查询对话框

**菜单路径**：工具 → 查询错误

**快捷键**：Ctrl + F

#### 4.2 查询方式

**关键字搜索**：
- 输入错误编号、描述或图层名称
- 支持模糊匹配

**条件筛选**：
- 按错误类型筛选
- 按处理状态筛选

**组合查询**：
- 可同时使用关键字和条件筛选

#### 4.3 查询结果

- 结果以表格形式显示
- 双击条目可定位到错误位置
- 点击"定位到错误"按钮跳转

---

### 五、视图操作

#### 5.1 缩放操作

| 操作 | 方式 |
|------|------|
| 放大 | 菜单"视图" → "放大" 或 Ctrl + + |
| 缩小 | 菜单"视图" → "缩小" 或 Ctrl + - |
| 适应窗口 | 菜单"视图" → "适应窗口" 或 F键 |
| 鼠标滚轮 | 滚轮向上放大，向下缩小 |

#### 5.2 平移操作

- 在非标记模式下，按住鼠标左键拖动
- 适用于查看大尺寸地图的不同区域

#### 5.3 坐标显示

状态栏实时显示鼠标位置的坐标信息。

---

### 六、统计功能

**菜单路径**：工具 → 统计信息

**统计内容包括**：
- 错误总数
- 按类型统计
- 按状态统计

## 常见问题

### Q1: 无法打开某些文件？

**可能原因**：
- 文件格式不支持
- 文件已损坏
- 文件被其他程序占用

**解决方法**：
- 确认文件格式在支持列表中
- 尝试用其他软件打开验证文件完整性
- 关闭其他可能占用文件的程序

### Q2: 程序启动后闪退？

**可能原因**：
- 缺少依赖库
- 配置文件错误

**解决方法**：
- 运行 `pip install -r requirements.txt` 安装依赖
- 删除config.yaml文件，程序会自动创建默认配置

### Q3: 错误标记无法保存？

**可能原因**：
- 没有写入权限
- 磁盘空间不足

**解决方法**：
- 以管理员身份运行程序
- 检查磁盘空间
- 尝试另存到其他位置

### Q4: 地图显示不正常？

**可能原因**：
- 图像尺寸过大
- 内存不足

**解决方法**：
- 使用较小的测试数据
- 关闭其他占用内存的程序
